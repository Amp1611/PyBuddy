## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Amp1611/PyBuddy.git
   cd PyBuddy
   ```

2. Install the required dependencies:
   ```bash
   pip install pandas
   ```

3. (Optional) Create a `riddles.csv` file with columns: `question`, `answer`, `hint` for the Riddles module. 

## Usage

Run the chatbot:
```bash
python CHATBOT. py
```

Follow the on-screen menu to select your desired learning activity. 

## Requirements

- Python 3.x
- pandas
- tkinter (usually included with Python)
